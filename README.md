# 📊 Streaming App Retention Analysis

## 📌 Problem Statement
The application has high downloads but low user retention, indicating a gap between acquisition and engagement.

---

## 🎯 Objective
Identify drop-offs in the user journey and improve retention using data-driven insights.

---

## 📈 Key Metrics
- DAU
- Retention Rate (D1, D7, D30)
- Churn Rate
- Session Time
- Conversion Rate

---

## 🔍 User Funnel
Download → Signup → Subscription → Content Consumption → Retention

---

## ⚠️ Key Insight
Significant drop-off observed between:
- Signup → Subscription  
- First Usage → Repeat Usage  

Users fail to perceive value early, leading to churn.

---

## 🧠 Root Causes
- Poor onboarding experience  
- Low perceived product value  
- Weak recommendations  
- Poor feature awareness  
- Technical issues  

---

## 💡 Solutions
- Simplified onboarding  
- Personalized recommendations  
- Improved engagement features  
- Push notifications  
- Technical optimizations  

---

## ⚙️ System Flow
Frontend → API → Backend → Database → Response

---

## 📈 Expected Impact
- Improved retention  
- Reduced churn  
- Better user engagement  

---

## 🏁 Conclusion
This project demonstrates how business analysis and user behavior insights can improve product retention and overall growth.
